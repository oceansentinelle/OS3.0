Ce que nous savons

  • La source actuelle est COAST-HF via SEANOE, DOI 10.17882/100119.
  • Les variables publiques disponibles pour BARAG_PROXY sont mesurées (MEASURED).
  • Les mesures affichées ici sont historiques (EXPIRED).
  • decision_ready=false implique un affichage non décisionnel.

Ce que nous ne savons pas

  • ERDDAP Ifremer est instable: réponses HTTP 503 observées.
  • Coriolis: timelines accessibles, mais les appels timeseries testés retournent 0 point exploitable.
  • Coriolis-Côtier /observations: catalogue de métadonnées, pas encore endpoint de valeurs exploitable.
  • L'incertitude est explicitement affichée lorsqu'elle existe; sinon elle reste UNKNOWN.
Limites publiques / Public limitations. FR: une mesure historique ne permet pas de conclure sur l'état présent du milieu. EN: an expired historical measurement cannot support an operational present-state decision.
