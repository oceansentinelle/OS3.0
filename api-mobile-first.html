Ocean Sentinel

API REST & Alertes SACS

Accédez aux données océanographiques en temps réel via notre API REST professionnelle. Recevez des webhooks pour les alertes critiques d'hypoxie et d'acidification.

Exemple d'Alerte Critique

⚠️ HYPOXIE DÉTECTÉE
Oxygène 135 µmol/kg < 150 µmol/kg
● MESURÉ Station BARAG • Risque mortalité huîtres

Cette alerte est envoyée automatiquement via webhook lorsque le seuil critique est franchi.

Fonctionnalités API

📊

Données Temps Réel

Accédez aux mesures des stations BARAG et EYRAC avec une latence < 10 minutes.

Endpoints Principaux

GET /api/v1/station/{station_id}/latest

Récupère les dernières mesures d'une station avec badges de vérité. 

{
  "station_id": "BARAG_PROXY",
  "timestamp": "2026-04-22T23:10:00Z",
  "parameters": [
    {
      "name": "TEMP",
      "value": 18.5,
      "unit": "°C",
      "status": "measured",
      "quality_flag": 1
    },
    {
      "name": "DOX2",
      "value": 135,
      "unit": "µmol/kg",
      "status": "measured",
      "is_critical": true
    }
  ]
}
GET /api/v1/alerts/sacs

Liste toutes les alertes SACS actives avec leur niveau de criticité. 

{
  "alerts": [
    {
      "id": "alert_hypoxie_001",
      "type": "HYPOXIE",
      "severity": "critical",
      "parameter": "DOX2",
      "value": 135,
      "threshold": 150,
      "station": "BARAG_PROXY",
      "status": "measured",
      "timestamp": "2026-04-22T23:08:00Z",
      "message": "Risque mortalité huîtres"
    }
  ]
}
POST /api/v1/webhooks/register

Enregistre un webhook pour recevoir les alertes en temps réel. 

{
  "url": "https://votre-serveur.com/webhook",
  "events": ["alert.critical", "alert.warning"],
  "secret": "votre_secret_webhook"
}

Règle de Vérité SACS

Chaque donnée retournée par l'API inclut un champ status indiquant sa provenance :

● MESURÉ Donnée directe capteur (fiabilité 100%)
◐ INFÉRÉ Proxy satellitaire Sentinel-3 (fiabilité 70-90%)
○ SIMULÉ Modèle MARS3D Ifremer (fiabilité 50-70%)
