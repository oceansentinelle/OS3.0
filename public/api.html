🚀 Endpoints Disponibles

GET /api/v1/health

Vérification de santé de l'API

Exemple : 
curl http://76.13.43.3:8000/api/v1/health
GET /api/v1/station/{station_id}/latest

Dernière mesure d'une station

Paramètres :
  • station_id : BARAG, VPS_PROD, etc.
Exemple : 
curl http://76.13.43.3:8000/api/v1/station/BARAG/latest
GET /api/v1/station/{station_id}/history

Historique des mesures

Paramètres :
  • station_id : Identifiant de la station
  • start_date : Date de début (ISO 8601)
  • end_date : Date de fin (ISO 8601)
  • limit : Nombre max de résultats (1-1000)
Exemple : 
curl "http://76.13.43.3:8000/api/v1/station/BARAG/history?limit=10"

💡 Exemples d'Utilisation

Python

import requests

# Récupérer le status
response = requests.get('http://76.13.43.3:8000/api/v1/health')
data = response.json()
print(f"Status: {data['status']}")

# Dernière mesure
response = requests.get('http://76.13.43.3:8000/api/v1/station/BARAG/latest')
measurement = response.json()
print(f"Température: {measurement['temperature_water']}°C")

JavaScript

// Récupérer le status
fetch('http://76.13.43.3:8000/api/v1/health')
  .then(response => response.json())
  .then(data => console.log('Status:', data.status));

// Dernière mesure
fetch('http://76.13.43.3:8000/api/v1/station/BARAG/latest')
  .then(response => response.json())
  .then(data => console.log('Température:', data.temperature_water + '°C'));

cURL

# Health check
curl http://76.13.43.3:8000/api/v1/health

# Dernière mesure avec formatage JSON
curl http://76.13.43.3:8000/api/v1/station/BARAG/latest | jq

# Historique 24h
curl "http://76.13.43.3:8000/api/v1/station/BARAG/history?limit=100" | jq
