Le Projet Ocean Sentinelle

Surveillance environnementale du Bassin d'Arcachon

Contexte

Le Bassin d'Arcachon est un écosystème marin fragile soumis à de multiples pressions : urbanisation, tourisme, aquaculture, changement climatique. La surveillance continue de ses paramètres océanographiques est essentielle pour anticiper les crises écologiques.

Objectifs

  • Centraliser les données océanographiques de sources multiples
  • Détecter automatiquement les anomalies environnementales
  • Alerter les professionnels en temps réel
  • Fournir une API REST pour l'intégration dans d'autres systèmes

Système d'Alertes

Détection automatique des seuils critiques :

  • •
    Acidification: pH < 7.8 (critique) ou < 7.9 (warning)
  • •
    Hypoxie: O₂ < 150 µmol/kg (critique) ou < 175 µmol/kg (warning)
  • •
    Eutrophisation: Chlorophylle-a > seuils définis
  • •
    Anomalies thermiques: Écarts significatifs par rapport aux moyennes saisonnières
