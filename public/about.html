🎯 Le Projet Ocean Sentinelle

Surveillance environnementale du Bassin d'Arcachon

📖 Contexte

Le Bassin d'Arcachon est un écosystème marin fragile soumis à de multiples pressions : urbanisation, tourisme, aquaculture, changement climatique. La surveillance continue de ses paramètres océanographiques est essentielle pour anticiper les crises écologiques.

🎯 Objectifs

  • Centraliser les données océanographiques de sources multiples
  • Détecter automatiquement les anomalies environnementales
  • Documenter les données publiques ou historiques disponibles
  • Séparer les scénarios des mesures et afficher les limites

🔬 Sources de Données

ERDDAP COAST-HF

Fréquence : Horaire

Stations : Arcachon Eyrac, Cap Ferret

Paramètres : Température, salinité, oxygène, pH, turbidité, chlorophylle-a

Source : Ifremer

Hub'Eau

Fréquence : Toutes les 6 heures

Zone : Bassin d'Arcachon

Paramètres : Qualité de l'eau, indicateurs biologiques

Source : BRGM / Agences de l'Eau

Scénarios en shadow mode

Les éléments ci-dessous sont des hypothèses de travail non décisionnelles :

  • Acidification : contexte à sourcer avant toute interprétation
  • Hypoxie : scénario proxy, non assimilable à une mesure
  • Eutrophisation : hypothèse documentaire, à sourcer
  • Anomalies thermiques : exploration scientifique sans décision automatique
