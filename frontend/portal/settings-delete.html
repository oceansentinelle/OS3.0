Suppression de compte

Attention : Cette action est irréversible et entraînera la perte définitive de toutes vos données.

Vous pouvez supprimer définitivement votre compte Ocean Sentinel à tout moment, conformément à votre droit à l'effacement (Art. 17 RGPD).

Que se passera-t-il ?

  • ✓ Suppression immédiate de votre compte
  • ✓ Révocation de toutes vos clés API
  • ✓ Suppression de toutes vos alertes configurées
  • ✓ Effacement de vos données personnelles sous 30 jours
  • ✓ Perte d'accès à l'historique de vos données

Délai d'effacement : Vos données seront effacées sous 30 jours maximum, conformément aux obligations légales de traçabilité (RGPD). Les données de facturation seront conservées 10 ans (obligation comptable française).
